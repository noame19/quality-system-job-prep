### 关键判定规律（重要）

- **"助理"** = 协助型岗，学历大专以上、0–1 年经验可投，主要跑腿 + 文档 + 数据录入
- **"文员"** = 几乎纯文档岗，要求最低，是你目前最强匹配
- **"专员"** = 中间岗，**要警惕**：
  - 写"体系专员/实习生 / 体系专员助理 / 注册专员"的 → 大概率可以投（实际等同于助理）
  - 写"体系专员"+ 客户审核/验厂/IATF16949 关键词的 → 实质是工程师，跳过
- **"工程师"** = 高要求岗，需要审厂、内审员证、主导经验，跳过

## 四、面试前必备通用理论（24 小时突击包）

### 4.1 ISO 9001 质量管理体系（**最重要，必背**）
- **七项质量管理原则**：以客户为中心、领导作用、全员参与、过程方法、改进、循证决策、关系管理
- **PDCA 循环**：Plan–Do–Check–Act，每一步做什么
- **体系文件四层结构**：
  - 一级：质量手册
  - 二级：程序文件（22 个标准程序）
  - 三级：作业指导书 SOP / WI
  - 四级：表单/记录
- **核心术语**：受控文件、有效文件、作废文件、外来文件、版本号、文件编号规则
- **文件全生命周期**：编制 → 审核 → 批准 → 编号 → 发放 → 使用 → 回收 → 作废 → 销毁

### 4.2 ISO 9001:2015 新版结构（**面试必问**）
- **10 个章节**（第 4–10 章是核心）：
  - 4.组织环境 5.领导作用 6.策划 7.支持 8.运行 9.绩效评价 10.改进
- **基于风险的思维**（Risk-based thinking）— 2015 版最大变化
- 删除了"预防措施"独立条款，并入"风险和机遇"

### 4.3 文件管控基本功（文控岗面试重点）
- **文件编号规则**：部门代码 + 文件类别 + 顺序号 + 版本号（如 QA-001-V1.0）
- **版本号表示**：V1.0 → V1.1 → V2.0（小改 vs 大改）
- **文件状态**：受控/非受控/作废/保密
- **分发方式**：原件留档、复印件盖章"受控副本"、电子件水印
- **借阅登记本**：日期/文件号/借阅人/归还日期/签名
- **电子档案备份**：本地 + 异地 / NAS / 云端，定期（日/周/月）

### 4.4 内审 / 外审基础（被问"协助审核怎么做"）
- **内审流程**：年度审核计划 → 审核方案 → 审核组组建 → 编制检查表 → 现场审核 → 不符合报告 → 整改验证 → 审核报告
- **不符合项分级**：严重不符合 / 一般不符合 / 观察项 / 改进建议
- **审核技巧**：抽样的逻辑（随机抽样 vs 重点抽样）、问"5 个为什么"（5WHY）
- **常见审核发现**：文件过期未回收、未做培训记录、未做设备校准、记录缺失

### 4.5 制造业基础常识（小白必知）
- **生产五大要素**：人机料法环（4M1E）
- **生产流程**：IQC（来料）→ 储存 → 上料 → 制程（IPQC）→ 装配 → 老化测试 → OQC（出货检验）→ 出货
- **QA vs QC 区别**（**面试必问**，要会讲清）：
  - QA = Quality Assurance **质量保证** = 建体系、做流程、防问题发生（**文员做的多**）
  - QC = Quality Control **质量控制** = 检验产品、找不良品（**检验员做的，要下车间**）
  - 一句话：QA 是造伞，QC 是淋雨后数雨滴
- **品质成本**：预防成本、鉴定成本、内部失败成本、外部失败成本
- **QC 七大手法**（文员偶尔被问）：查检表、柏拉图、鱼骨图、散布图、直方图、控制图、层别法
- **ROHS 2.0**（电子类必知）：铅、汞、镉、六价铬、PBB、PBDE、DEHP/DBP/BBP 四种邻苯，共 **10 项**

### 4.6 文员岗实务（Office + ERP）
- **Word**：样式、目录生成、页眉页脚、修订模式、模板套用
- **Excel**：数据透视表、VLOOKUP、COUNTIF、SUMIF、条件格式、冻结窗格、台账模板
- **ERP 系统概念**（你不会但要懂）：BOM（物料清单）、ECN（工程变更通知）、物料编码
- **文件分类命名**：日期+部门+内容+版本（如 20260624-QA-不良品周报-V1.0）
- **记录保存年限**：质量记录 ≥ 3 年，产品生命周期 + 1 年（部分行业）

### 4.7 面试高频问题预判（建议提前写好答案）
1. 你对 ISO 9001 了解多少？新版 2015 的核心变化是什么？
2. QA 和 QC 的区别是什么？
3. 文件版本控制怎么做？文件受控状态有几种？
4. 你怎么处理过期文件？怎么防止现场用错版本？
5. 你没做过体系工作，你觉得自己凭什么能胜任？（**用 EHS 经验迁移**）
6. 加班怎么看？（**明确接受偶尔加班**，但主动表态高效完成本职工作）
7. 你对 AC 电源线这个行业了解吗？（**背第三部分小家电/电源线工艺**）

- **产品结构**：导体（铜/镀锡铜）+ 绝缘层（PVC/PE）+ 护套 + 插头（国标/欧标/美标/英标）
- **关键认证**：CCC（中国强制）、UL（美国）、CE（欧盟）、VDE（德国）、PSE（日本）、KC（韩国）
- **关键测试项目**：
  - 耐压测试（Hi-Pot）
  - 绝缘电阻
  - 导体电阻
  - 阻燃测试（VW-1、FT2）
  - 摇摆测试（Flexing test，电源线寿命）
  - 温升测试
- **ROHS 2.0**：10 项有害物质（电子制造业必知）
- **ECN（工程变更）**：插头规格变更、线材长度变更、材料替代
- **BOM 概念**：电源线的 BOM 包含线材、插头、尾卡、扎带、包装

### 五大变化（2015 vs 2008）

1. 更少规定性要求
2. "成文信息"统一文件+记录

### 助动词含义

- **应** = 要求（缺失 = 不符合）
- **宜** = 建议（缺失不扣分）
- **可** = 允许
- **能** = 可能

### 三大支柱

- **过程方法**（Process Approach）— 4.4 / 8.1
- **PDCA**（Plan-Do-Check-Act）
- **基于风险的思维**（Risk-Based Thinking）— 6.1

### 七项质量管理原则（标准译法）

1. 以顾客为关注焦点
2. 领导作用
3. 全员积极参与
4. 过程方法
5. 改进
6. 循证决策
7. 关系管理

### 6.1 应对风险 5 种方式

- 规避
- 改变可能性
- 改变后果
- 分担
- 通过信息充分决策保留

### 10.2.1 纠正措施 6 步循环

a) 应对 → b) 分析 → c) 措施 → d) 评审有效性 → e) 更新风险表 → f) 变更 QMS
