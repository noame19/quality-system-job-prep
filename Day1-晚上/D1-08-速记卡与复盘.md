## 🟦 速记卡 1：ISO 9001 七项质量管理原则

```
1. 以客户为中心（关注客户需求和期望）
2. 领导作用（最高管理者带头）
3. 全员参与（每个人都是质量责任人）
4. 过程方法（把工作拆成一个个过程管）
5. 改进（持续改进 / PDCA）
6. 循证决策（用数据说话，不用感觉）
7. 关系管理（供应商/客户双赢）

口诀：客领全过改循关（kè lǐng quán guò gǎi xún guān）
```

---

## 🟦 速记卡 2：PDCA 戴明环

```
P  Plan   策划：定目标、定方法
D  Do     执行：按计划做
C  Check  检查：对照目标看效果
A  Act    改进：巩固/修正

对应 ISO 9001:
  P → 第 4、5、6 章
  D → 第 7、8 章
  C → 第 9 章
  A → 第 10 章
```

## 🟦 速记卡 5：文件编号规则

```
格式: AA-BBB-CCCC-D.E
AA   公司代码
BBB  部门代码 (QA / QC / PRD / ENG / HR ...)
CCCC 文件类别 (PR/SOP/WI/FRM/SPEC/ECN ...)
D    顺序号 (001-999)
E    版本号 (V1.0)

例子: QA-SOP-005-V1.0
      = 品保部标准作业程序第5份第1版
```

## 🟦 速记卡 7：QA vs QC 对比

| | QA（质量保证） | QC（质量控制） |
|---|---|---|
| 性质 | 建体系 / 定规则 | 检验产品 / 找不良 |
| 阶段 | 事前 + 事中 + 事后 | 事后（为主） |
| 工具 | SOP/标准/审核 | 卡尺/仪器/抽检 |
| 岗位 | 文员、体系工程师 | 检验员（要下车间） |
| 一句话 | **造伞** | **淋雨后数雨滴** |

> ⚠️ 你要投 QA 岗，不要投 QC 自检岗

## 🟦 速记卡 8：QC 七大手法

```
1. 查检表  Check Sheet    - 数据收集
2. 柏拉图  Pareto         - 抓重点（80/20）
3. 鱼骨图  Fishbone       - 找原因（5M1E）
4. 直方图  Histogram      - 看分布
5. 散佈图  Scatter        - 看相关
6. 控制图  Control Chart  - 看异常
7. 层别法  Stratification - 分层解析

口诀：查柏鱼直散控层
```

## 🟦 速记卡 9：ROHS 2.0 十项有害物质

```
1. 铅 (Pb)          - 焊料、玻璃
2. 镉 (Cd)          - 开关、连接器
3. 汞 (Hg)          - 温控器、传感器
4. 六价铬 (Cr VI)    - 金属防腐蚀涂层
5. 多溴联苯 (PBB)    - 阻燃剂
6. 多溴二苯醚 (PBDE) - 阻燃剂
7. 邻苯二甲酸二异丁酯 (DIBP)
8. 邻苯二甲酸二丁酯 (DBP)
9. 邻苯二甲酸丁苄酯 (BBP)
10. 邻苯二甲酸二己酯 (DEHP)

浓度限值: Cd ≤ 100ppm；其他 ≤ 1000ppm
```
