## 二、纸质档案管理

### 1. 档案室基本要求

| 项目 | 标准 |
|---|---|
| 位置 | 独立房间，防火、防潮、防虫、防盗 |
| 温湿度 | 温度 14–24℃，湿度 45–60% |
| 消防 | 配置灭火器（气体灭火优先） |
| 照明 | 防紫外线灯具 |
| 货架 | 金属密集架或档案柜 |
| 准入 | 授权人员进出登记 |

### 2. 档案分类（一级 / 二级 / 三级）

### 3. 档案盒命名规则（贴在档案盒侧面）

```
格式: AABB-CCCC
AA    : 一级分类代码 (01-07)
BB    : 二级分类代码 (01-99)
CCCC  : 年份+月份 (YYYYMM)

示例:
  0101-202401  → 体系/质量手册/2024年1月
  0301-202406  → 品质/IQC/2024年6月
  0305-202405  → 品质/CAPA/2024年5月
```

### 4. 档案盒内文件排列

- 按时间顺序排列（最新的在最上面）
- 文件之间放分隔纸（标注小类）
- 盒内附《档案目录表》

### 2. 文件命名规则（电子）

```
格式: [状态]_编号_名称_版本_日期_部门_版本人.扩展名
示例: 
  [受控]_QA-SOP-005_来料检验SOP_V1.0_20240620_QA_张三.docx
  [作废]_QA-SOP-005_来料检验SOP_V0.9_20240315_QA_李四.docx
  [非受控]_ENG-SPEC-001_AC电源线规格书_V3.0_客户参考.pdf
```

### 3. 访问权限（**关键**）

| 用户组 | 可访问 | 可修改 |
|---|---|---|
| 文控员 | 全部 | 仅文件目录 |
| 部门经理 | 本部门 | 本部门文件 |
| 工程师 | 本部门 | 本部门 SOP |
| 普通员工 | 仅与自己相关 | ❌ |
| 外部人员 | 仅"非受控" | ❌ |

## 四、备份策略（**3-2-1 + 异地 + 加密**）

### 1. 3-2-1 原则

- **3** 份副本（原件 + 2 个备份）
- **2** 种介质（如：服务器 + NAS；或服务器 + 云）
- **1** 份异地（不在同一栋楼）

### 2. 备份频率

| 数据级别 | 备份频率 | 保留期 |
|---|---|---|
| 关键受控文件 | **实时同步** | 永久 |
| 品质记录 | **每日备份** | ≥ 3 年 |
| 一般文件 | **每周备份** | ≥ 1 年 |
| 系统数据（ERP/邮箱） | **每日备份** | ≥ 5 年 |

### 3. 备份介质

- **磁盘阵列**（服务器本地）
- **NAS**（网络附加存储）
- **云端**（阿里云盘 / OneDrive 企业版 / 百度网盘企业版）
- **离线移动硬盘**（每周五做一次，异地保管）

### 4. 备份验证

> **备份 ≠ 一定能恢复**！每月抽 1 个文件做恢复演练。

## 五、档案保存期限（**重要**）

| 文件类型 | 保存期限 |
|---|---|
| 质量手册 / 程序文件 | 永久（直到被新版替代 + 5 年） |
| SOP / WI | 现行 + 旧版保留 ≥ 3 年 |
| 品质记录（IQC/IPQC/OQC） | ≥ 3 年（部分 ≥ 5 年） |
| RoHS 测试报告 | ≥ 5 年（部分 ≥ 10 年） |
| 不合格品报告 / CAPA | ≥ 5 年 |
| 内审 / 外审报告 | ≥ 5 年 |
| 管理评审记录 | ≥ 5 年 |
| 培训记录 | 员工在职 + 离职后 3 年 |
| 客户投诉记录 | ≥ 5 年 |
| 设备校准证书 | 设备使用期 + 5 年 |
| ECN 工程变更 | 永久 |

## 六、档案销毁流程

```
保留期满
    ↓
文控岗提出销毁申请
    ↓
部门主管审核（确认无追溯需求）
    ↓
体系经理审批
    ↓
监销人监督销毁
    ↓
填写《档案销毁记录》并归档
    ↓
保留销毁记录 ≥ 5 年
```

## 八、面试杀手锏：档案管理的 3 个亮点（讲给 HR 听）

1. **电子档案 + 纸质档案双轨制**：纸质作法律凭证，电子作日常查阅
2. **3-2-1 备份 + 月度恢复演练**：备份不等于安全
3. **权限分级 + 借阅登记**：信息安全和效率兼顾

## 十、自测题

1. 档案分类的一级目录至少分几类？
2. 备份 3-2-1 原则的"3-2-1"分别指什么？
3. 品质记录的保存期一般 ≥ 几年？
4. 借阅登记 Excel 必须包含哪些字段？

**参考答案**：
1. 至少 6 类（体系/技术/品质/行政/人事/客户供应商）
2. 3 份副本、2 种介质、1 份异地
3. ≥ 3 年
4. 序号、日期、档案编号、名称、借阅人、部门、目的、计划归还、实际归还
