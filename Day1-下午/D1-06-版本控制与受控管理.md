## 一、版本控制核心原则

> **确保现场任何时候只有一个版本在用**——这是文控岗存在的根本价值。

---

## 二、版本号规则（详细版）

### 1. 版本号表示法

**国际通用格式**：`V<主版本号>.<次版本号>`

| 变化类型 | 版本号变化 | 触发条件 | 例子 |
|---|---|---|---|
| **小改（小版本）** | 次版本 +0.1 | 文字/格式/数据修正；增加附录；删错别字 | V1.0 → V1.1 |
| **大改（大版本）** | 主版本 +1.0，次版本归 0 | 章节/工艺/流程/法规变化；结构性变更 | V1.9 → V2.0 |
| **首次发布** | V1.0 | 新文件 | — |

**反例（绝对不能这么干）**：
- ❌ V1.0 → V1.5（中间改了 5 次但没记录）
- ❌ V1.0 → V2.0 但其实只改了错别字
- ❌ 不同步更新总清单

### 2. 升版必须走的流程

```
业务变更（ECN/工艺变更/标准更新）
    ↓
起草新版（升版本号）
    ↓
审核 + 批准
    ↓
【文控岗】总清单更新版本
    ↓
新版发放 + 旧版回收 + 旧版作废
    ↓
现场使用新版
```

**关键点**：**新版本发放前，旧版本就要全部收回**。不允许出现"新版本已发，旧版本还在现场"的情况。

## 三、受控标识（3 套方案）

### 方案 A：纸质受控章（最传统）
```
   ┌────────────────┐
   │   [受控副本]   │    蓝色公章（与红色公章区分）
   │   编号：C-001  │    编号：与文件总清单一致
   │   发放：生产部  │    发放范围：哪个部门用
   │   生效：2024-01 │    生效日期：批准日
   └────────────────┘
```
**盖章位置**：首页右上角 + 每一页的页脚。

### 方案 B：受控标签（贴纸式）
- 黄色 / 蓝色标签纸，打印后贴
- 适合批量发放

### 方案 C：电子文件水印
- **PDF 水印**：在每页加"受控副本 · 严禁外传 · 编号 xxx"
- **Word 文档**：页眉加"受控副本"
- **共享盘**：在文件名加 `[受控]` 前缀

---

## 四、受控状态分类（4 类）

| 状态 | 标识 | 用途 |
|---|---|---|
| **受控** | 蓝色受控章 | 公司内部使用 |
| **非受控** | 红色"仅供参考" | 客户、供应商、外部参考 |
| **作废** | 红色"作废" + 日期 | 已废止，保留备查 |
| **保密** | 加"机密/秘密/绝密" + 加密 | 受密级管控 |

## 五、借阅与归还登记（必备台账）

### 1. 纸质借阅登记本

| 日期 | 文件编号 | 文件名 | 借阅人 | 借阅用途 | 计划归还日 | 实际归还日 | 经手人 |
|---|---|---|---|---|---|---|---|
| 2024-06-20 | QA-PR-001 | 检验管理程序 | 张三 | 客户验厂准备 | 2024-06-22 | 2024-06-22 | 李四 |
| 2024-06-23 | ENG-SPEC-001 | AC 电源线规格书 | 王五 | 新供应商开发 | 2024-06-30 | 2024-07-02 | 李四 |

**要点**：
- 借阅期限一般 ≤ 7 天（紧急 ≤ 24 小时）
- 超期未还：文控岗催还
- 复印必须文控岗授权
- 借阅人不得私自给第三方

### 2. 电子文件借阅

- 通过 ERP/PLM 系统申请 → 审批 → 系统自动记录
- 权限到期自动收回

## 六、电子文件管控要点

### 1. 电子文件的命名规则（建议）

```
格式: [受控状态]_文件编号_文件名称_版本_日期_部门.扩展名
示例: [受控]_QA-SOP-005_来料检验SOP_V1.0_20240620_QA.docx
```

### 2. 电子文件的存储位置

```
公司服务器 (主)
    ├── /01_受控现行/        ← 最新有效版本
    ├── /02_作废/            ← 已废止版本（备查）
    ├── /03_外来文件/        ← 外部标准、法规
    ├── /04_模板/            ← 文件模板
    └── /99_备份/            ← 每日自动备份

个人电脑 (辅)
    └── 严禁保存受控文件唯一版本（必须从服务器读）
```

### 3. 备份策略 3-2-1 原则

- **3** 份副本
- **2** 种介质（如服务器 + 移动硬盘）
- **1** 份异地（云端 / 另一栋楼）

### 4. 电子文件水印（推荐）

```python
# 推荐用 Python 给 PDF 加水印
import fitz
doc = fitz.open("xxx.pdf")
for page in doc:
    page.insert_text((50, 30), "受控副本 · 编号QA-SOP-005 · 严禁外传",
                     fontsize=10, color=(0, 0, 1))
doc.save("xxx_受控.pdf")
```

## 七、过期文件"零容忍"管控（**面试高频**）

> **场景**：外审员走进车间，看到操作工拿着一份 SOP，问："这是最新版本吗？"

**正确做法**：
1. 立即收回过期文件
2. 现场张贴最新版本
3. 写《不符合项报告》：描述+立即整改+根本原因+预防措施
4. 跟踪整改效果
5. 纳入下一轮内审重点

**预防机制**：
- 每周现场巡检（文控岗）
- 文件版本变更 24 小时内完成回收
- 文件总清单每月更新 + 邮件推送
