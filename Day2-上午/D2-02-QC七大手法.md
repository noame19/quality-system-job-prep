## 一、QC 七大手法（**老七法**）

QC Seven Tools，又叫"老 QC 七大手法"或"QC 七工具"，由日本科技联盟于 1972 年提出。

> **核心作用**：用最简单的图表，把"数据"变成"信息"，帮我们发现问题、找出原因、追踪改善。

| # | 中文 | 英文 | 一句话用途 |
|---|---|---|---|
| 1 | 查检表 | Check Sheet | **收集**数据 |
| 2 | 层别法 | Stratification | **分类**数据 |
| 3 | 柏拉图 | Pareto Chart | **抓重点**（80/20） |
| 4 | 鱼骨图 | Fishbone / Ishikawa | **找原因** |
| 5 | 直方图 | Histogram | **看分布** |
| 6 | 散佈图 | Scatter Diagram | **看相关** |
| 7 | 控制图 | Control Chart | **看异常** |

**口诀**：**查柏鱼直散控层**（chá bǎi yú zhí sàn kòng céng）

## 二、每个手法详解（含应用场景）

### 1. 查检表（Check Sheet）—— 数据收集
**作用**：用简单的表格，把数据记录下来。
**例子**：
```
电源线不良品查检表
┌────────┬─────┬─────┬─────┬─────┐
│ 日期   │短路 │开路 │外被 │插头 │
├────────┼─────┼─────┼─────┼─────┤
│ 06-20  │  3  │  1  │  2  │  0  │
│ 06-21  │  5  │  2  │  1  │  1  │
└────────┴─────┴─────┴─────┴─────┘
```

### 2. 层别法（Stratification）—— 分类数据
**作用**：把数据按类别分开看，避免"平均数陷阱"。
**例子**：电源线不良率 5%（看起来还行），但按产线分层：A 线 1%，B 线 12% → 找 B 线原因。

### 3. 柏拉图（Pareto Chart）—— 80/20 抓重点
**作用**：找"最重要的问题"——少数原因造成多数问题（80/20 法则）。
**例子**：
```
不良原因                不良数    累计%
短路                    45       45%  ████████████
插头不良                 25       70%  ███████
外被划伤                 15       85%  ████
标识错误                  8       95%  ██
其他                      5      100%  █
```
**结论**：前 3 项占 85%，先解决短路、插头、外被。

### 4. 鱼骨图（Fishbone / Ishikawa）—— 找原因
**作用**：把一个"问题（结果）"的所有可能原因按类别列出来，像鱼骨一样。
**5M1E 类别**：
- **人（Man）**：员工技能、培训
- **机（Machine）**：设备、工装
- **料（Material）**：原材料、外购件
- **法（Method）**：工艺、作业方法
- **测（Measurement）**：测量方法、仪器
- **环（Environment）**：温湿度、噪声、照明

**画法**：
```
                人        机       料
                 \        |        /
                  \       |       /
                   \      |      /
        ┌──────────┬──────┬──┴────────┐
        │          │      │  问题    │
   法 ──┤ 鱼骨图   │  ←──  │ 短路多发 │
        │          │      │          │
        └──────────┴──────┴──┬────────┘
                   /      |      \
                  /       |       \
                 /        |        \
                测        环
```

### 5. 直方图（Histogram）—— 看分布
**作用**：看数据的"形状"（是否正态分布、是否偏移）。
**例子**：电源线导体直径测量值（单位 mm）
```
数量
  │  ▄
  │  █ ▄
  │  █ █ ▄
  │  █ █ █ ▄
  └─────────────────── 直径
     1.8 1.9 2.0 2.1 2.2
```

### 6. 散佈图（Scatter Diagram）—— 看相关
**作用**：看两个变量是否有关系。
**例子**：插头温度 vs 寿命
```
寿命
  ↑
  │      ●●
  │   ●●
  │ ●●●
  │●●
  └─────────→ 温度
   (正相关)
```

### 7. 控制图（Control Chart）—— 看异常
**作用**：监控过程是否"受控"。
**三根线**：
- **UCL**（上控制限）= μ + 3σ
- **CL**（中心线）= μ
- **LCL**（下控制限）= μ − 3σ

```
数量
  │   UCL ─────
  │
  │        ●   ← 异常点（超出 UCL）
  │
  │  CL ────── ──●─●──●─●──●─●─
  │
  │   LCL ─────
  └─────────────────── 时间
```

**异常判断**：
- 1 个点超出 UCL/LCL
- 7 个连续点在中心线上方或下方（链）
- 7 个连续点连续上升或下降（趋势）
- 2/3 的点超出 ±2σ 警戒区

## 三、文控岗会被问到 QC 七大手法吗？

**答**：**偶尔会被问到**（特别是品质文员 / QA 文员岗）。但**不会被深入考**——能说出名字和用途即可。

### 典型面试题

> Q1：QC 七大手法是哪些？（**必背**）
> Q2：柏拉图的原理？（80/20 法则）
> Q3：你用过哪个手法？（**建议说柏拉图或鱼骨图**，结合不良品分析举例）

### 准备答案模板

> "我学过 QC 七大手法。在 EHS 工作中，我用柏拉图分析过'现场隐患类别占比'，发现前 3 类（设备防护缺失、PPE 未佩戴、标识缺失）占 78% 的隐患，于是我们集中整改这三类，3 个月内隐患总数下降了 40%。"
