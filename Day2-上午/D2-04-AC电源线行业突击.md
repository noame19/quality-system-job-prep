## 一、AC 电源线是什么

- **AC = Alternating Current（交流电）**
- **用途**：家用电器、办公设备、电动工具外接电源线
- **电压**：通常 220V（国标）/ 110V（美标）/ 100-240V（全球通用）
- **典型产品**：电饭煲电源线、电脑电源线、电钻电源线

## 二、产品结构（**必背**）

```
电源线剖面图（剖开看）：

   ┌─────────────────────────────────┐
   │         外被 / 护套              │  ← 第 4 层：黑色/白色 PVC
   │   ┌─────────────────────────┐   │
   │   │       绝缘层（PVC/PE）   │   │  ← 第 3 层：彩色 PVC
   │   │   ┌─────────────────┐   │   │
   │   │   │   导体（铜丝）   │   │   │  ← 第 2 层：多股铜丝
   │   │   │  (镀锡铜/裸铜)   │   │   │
   │   │   └─────────────────┘   │   │
   │   └─────────────────────────┘   │
   └─────────────────────────────────┘
            │
            └──────→  插头（一体注塑或组装）
```

| 部件 | 材料 | 关键参数 |
|---|---|---|
| **导体** | 多股镀锡铜丝 / 裸铜 | 截面积 0.75mm² / 1.0mm² / 1.5mm²（家用通常 0.75mm²） |
| **绝缘层** | PVC（聚氯乙烯） | 颜色：棕 L / 蓝 N / 黄绿 PE；厚度 ≥ 0.6mm |
| **护套** | PVC | 黑色或白色；厚度 ≥ 0.8mm |
| **插头** | ABS / PVC + 铜片 | 国标 / 欧标 / 美标 / 英标 |

**常见型号**（线材标识）：
- **H05VV-F 3G 0.75mm²**：H = 欧规；05 = 300V/500V；VV = 双层 PVC；F = 细铜丝；3G = 3 芯带地线；0.75mm² = 单芯截面积
- **NISPT-2**：双绝缘扁线（无地线）
- **SPT-1W**：单绝缘扁线
- **SJTW**：防水电源线
- **SJTO**：耐油电源线

## 四、关键测试项目（**必背 5 项**）

### 1. 耐压测试（Hi-Pot / Withstand Voltage）
- **目的**：检测绝缘是否能承受高电压
- **方法**：线-线间 + 线-地间，加 **1500V AC / 1 分钟**（家用）或 **2000V AC**（工业）
- **判定**：不应击穿 / 闪络 / 漏电超标

### 2. 绝缘电阻（Insulation Resistance）
- **目的**：检测绝缘完整性
- **方法**：加 500V DC，测电阻
- **判定**：≥ 100 MΩ（多数标准）

### 3. 摇摆测试（Flexing Test）
- **目的**：模拟插头在使用中的弯折
- **方法**：固定插头，线材加重物（500g / 1kg），以一定角度反复弯折（如 ±60°）
- **判定**：通常 ≥ 1 万次 / 3 万次 / 10 万次（按产品规格定）

### 4. 阻燃测试（Flame Test）
- **目的**：检测阻燃性能
- **方法**：VW-1（垂直燃烧）、FT2（水平燃烧）
- **判定**：30 秒内自熄，不引燃脱脂棉

### 5. 温升测试（Temperature Rise Test）
- **目的**：检测满载时线材温度
- **方法**：通额定电流 + 1.25 倍，测 6 小时
- **判定**：温升 ≤ 50℃（不同标准不同）

## 六、ECN（工程变更通知）流程（**文控岗高频**）

```
ECN 触发原因:
  - 客户要求（如 Walmart 要求线材加粗到 1.0mm²）
  - 法规变化（如 RoHS 新增物质）
  - 成本优化（如换更便宜的 PVC 供应商）
  - 质量问题（如某个插头不良率高）

ECN 流程:
  1. 申请（工程师）→ 填《ECN 申请单》
  2. 评审（跨部门）→ 品质 / 采购 / 生产 / 销售会签
  3. 批准（工程经理 / 总经理）
  4. 执行
       - 更新规格书（SOP/SPEC）
       - 更新 BOM（ERP）
       - 更新工艺
       - 更新 RoHS 报告
  5. 【文控岗】编号发放 + 旧版回收
  6. 首批试产 + 验证（OQC）
  7. 量产
```

**文控岗在 ECN 中的关键动作**：
- 给 ECN 编号（ECN-2024-001）
- 同步更新《文件总清单》
- 收回所有受影响文件的旧版
- 把 ECN 通知下发到各部门
- 跟踪 ECN 执行情况

## 七、生产工艺流程（**AC 电源线**）

```
1. 拉丝（铜杆 → 细铜丝）
       ↓
2. 镀锡（部分需要，防氧化）
       ↓
3. 绞线（多股铜丝绞合）
       ↓
4. 挤绝缘（内层 PVC）
       ↓  ← 在线火花检测
5. 挤护套（外层 PVC）
       ↓  ← 在线火花检测
6. 印刷（线材印字：型号/规格）
       ↓
7. 裁线（按订单长度）
       ↓
8. 剥皮（剥掉两端绝缘/护套，露出铜丝）
       ↓
9. 端子压接（插头铜片压接，或注塑）
       ↓
10. 插头注塑（一体成型）或装配
       ↓
11. RoHS 抽样检测
       ↓
12. 电气性能测试（耐压 / 绝缘电阻 / 极性 / 接地连续性）
       ↓
13. 外观检查（OQC）
       ↓
14. 摇摆测试（部分高端产品）
       ↓
15. 老化测试（通电 1-4 小时）
       ↓
16. 包装 / 装尾卡
       ↓
17. 入库 / 出货
```
