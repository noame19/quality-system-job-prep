## 🎓 学习方法建议

### 1. 时间分配（80/20 原则）
- **80% 时间给 ISO 9001 + 文件管控**（你最大短板）
- **20% 时间给行业知识 + 模拟面试**

### 2. 主动产出（不是只看）
- 每节课件都按"输出物"做一份
- 用 Word/Excel 实际做出来（不要只看不练）

### 3. 复述输出
- 每学完一节，关上文档，用自己的话讲一遍
- 录音 → 回放 → 看哪里卡壳 → 重点复习

### 4. 面试前一天晚上
- 把速记卡 10 张过 3 遍
- 把 30 道面试题各答 1 次
- 把面试作品装进文件夹
