# D2-05｜内审 / 外审全流程（体系审核怎么"配合"）

> ⏱ 时长：90 min（14:00–15:30）
> 🎯 目标：能讲清内审 7 步流程 + 知道文控岗在审核中做什么
> 📎 输出物：一份"内审检查表"模板

---

## 一、什么是审核

| 类型 | 主体 | 目的 | 频率 |
|---|---|---|---|
| **第一方审核（内审）** | 公司自己审自己 | 检查体系是否有效运行 | 每年 1 次全覆盖 |
| **第二方审核（客户验厂）** | 客户来审 | 评估是否能成为供应商 | 按客户要求 |
| **第三方审核（外审）** | 认证机构（SGS/BSI/TÜV） | 颁发 / 监督认证证书 | 每年 1 次监督 + 3 年 1 次复评 |

---

## 二、内审 7 步全流程（**必背**）

```
① 年度审核计划
       ↓
② 审核方案 + 审核组组建
       ↓
③ 编制审核检查表
       ↓
④ 现场审核（首次会议 + 现场 + 末次会议）
       ↓
⑤ 不符合项报告（NCR）
       ↓
⑥ 整改 + 验证
       ↓
⑦ 审核报告 → 输入管理评审
```

---

## 三、每步详解

### ① 年度审核计划（年度审核方案）
- **时间**：每年 12 月制定下一年度
- **内容**：覆盖 ISO 9001 所有条款（4–10 章）+ 所有部门
- **要素**：
  - 审核目的
  - 审核准则（标准 + 文件）
  - 审核范围
  - 审核频次
  - 审核日期 / 审核员 / 受审部门

**文控岗动作**：把《年度审核计划》存档，并发放到相关部门。

### ② 审核方案 + 审核组组建
- **审核组长**：有内审员证的人
- **审核员**：每组 2-3 人，覆盖不同部门
- **文控岗**：作为"陪同员"或"记录员"参与

**内审员要求**：
- 接受过 ISO 9001 内审员培训（一般 16-24 学时）
- 通过考试（部分公司自己出题）
- 每年至少参加 1 次内审

### ③ 编制审核检查表
- **格式**：按 ISO 9001 条款 + 部门
- **示例**（文控岗配合部分）：

```
┌─────────────────────────────────────────────────────────┐
│ ISO 9001 内审检查表（品保部 / 文控）                       │
├──────┬──────────────┬──────────────┬──────────┬─────────┤
│ 条款 │ 审核要点     │ 查证方法     │ 审核发现 │ 判定    │
├──────┼──────────────┼──────────────┼──────────┼─────────┤
│7.5.1 │ 是否建立文件 │ 查程序文件   │ 有 PR-001│ 符合    │
│      │ 化信息       │              │          │         │
├──────┼──────────────┼──────────────┼──────────┼─────────┤
│7.5.2 │ 文件创建更新 │ 抽查 5 份文件 │ V2.0 流程│ 符合    │
│      │ 是否规范     │ 看版本号     │          │         │
├──────┼──────────────┼──────────────┼──────────┼─────────┤
│7.5.3 │ 受控标识     │ 看是否有蓝章 │ 都有     │ 符合    │
├──────┼──────────────┼──────────────┼──────────┼─────────┤
│7.5.3 │ 现场是否用   │ 现场抽查 3 处 │ 现场全是 │ 符合    │
│      │ 最新版本     │ SOP          │ V2.0     │         │

│7.5.3 │ 旧版是否回收 │ 查回收登记   │ 已回收   │ 符合    │
├──────┼──────────────┼──────────────┼──────────┼─────────┤
│ ...  │ ...          │ ...          │ ...      │ ...     │
└──────┴──────────────┴──────────────┴──────────┴─────────┘
```

### ④ 现场审核（**重点**）

#### 4.1 首次会议（15 分钟）
- 审核组长主持
- 介绍审核组成员、审核目的、范围、日程
- 部门负责人出席

#### 4.2 现场审核（2-4 小时 / 部门）
**审核技巧**：
- **抽样**：随机抽 3-5 份记录 / 现场抽 3-5 处
- **问 5W2H**：What / Who / When / Where / Why / How / How much
- **顺向追踪**：从客户订单 → 评审 → 生产 → 出货 → 客后
- **逆向追踪**：从不合格品 → CAPA → 整改 → 关闭
- **5Why 分析**：一个问题连续问 5 个"为什么"找根本原因

#### 4.3 末次会议（15 分钟）
- 审核组长通报审核发现
- 宣读不符合项报告
- 与受审部门确认

### ⑤ 不符合项报告（NCR）
**3 类不符合**：

| 类型 | 严重程度 | 例子 |
|---|---|---|
| **严重不符合** | Major | 体系完全失效；多个条款违反 |
| **一般不符合** | Minor | 单点不符合 |
| **观察项 / 改进建议** | Observation | 潜在风险 |

**NCR 模板**：
```
不符合项报告 (NCR)
编号: NCR-2024-001
审核日期: 2024-06-25
受审部门: 仓库
条款: 7.5.3 文件控制
描述: 仓库现场使用的 IQC SOP 是 V1.0 版本，但现行有效版本为 V1.1（2024-05-10 生效）
不符合标准: ISO 9001:2015 7.5.3.2
不符合事实: 现场文件过期未回收
审核员: 张三          日期: 2024-06-25
部门负责人确认: 李四   日期: 2024-06-25
```

### ⑥ 整改 + 验证（**30 天内**）

**整改流程**：
```
NCR 发出
    ↓
部门原因分析（5Why + 鱼骨图）
    ↓
制定纠正措施（短期 + 长期）
    ↓
实施整改
    ↓
提交证据（照片 / 新文件 / 培训记录）
    ↓
审核员验证有效性
    ↓
关闭 NCR 或退回重改
```

### ⑦ 审核报告
- **内容**：审核目的、范围、准则、发现、结论、建议
- **存档**：作为管理评审输入
- **文控岗动作**：归档到"内审"文件夹

## 四、外审 / 客户验厂的"文控配合清单"

### 1. 提前 1 周准备

- [ ] 文件总清单（最新版）
- [ ] 内审报告 + 管理评审记录
- [ ] 不符合项整改记录（全部关闭）
- [ ] 认证证书原件 + 复印件
- [ ] RoHS / CCC / CE 证书
- [ ] 主要客户清单 + 投诉记录

### 2. 审核当天配合

- [ ] 会议室准备（投影、白板、水）
- [ ] 各部门陪同员到位
- [ ] 现场 5S（整洁有序）
- [ ] 现场文件全部受控、最新版本

### 3. 审核结束后

- [ ] 不符合项确认签字
- [ ] 整改承诺时间
- [ ] 跟踪整改
- [ ] 拿到审核报告

---

## 五、文控岗在审核中的"3 大任务"

1. **陪同**：全程陪同外审员，回答文件相关问题
2. **记录**：实时记录审核员的提问和发现
3. **应对**：现场抽文件时 30 秒内调出（靠文件总清单 + 借阅登记）
