## 二、Word 必学 6 个技能

### 1. 样式（最基础、最重要）
```
操作路径: 开始 → 样式区
常用样式:
  - 标题 1 (一级标题, 黑体, 16pt)
  - 标题 2 (二级标题, 黑体, 14pt)
  - 标题 3 (三级标题, 黑体, 12pt)
  - 正文 (宋体, 12pt)
  - 页眉页脚

好处: 改一处样式, 全文统一变
```

### 2. 自动目录
```
操作路径: 引用 → 目录 → 自动目录
前提: 必须先给标题应用"标题 1/2/3"样式
目录会自动从标题生成, 内容变了右键"更新域"
```

### 3. 页眉页脚（**文控岗最常用**）
```
操作路径: 插入 → 页眉 / 页脚
内容建议:
  页眉左: 公司 Logo + 文件名
  页眉右: 文件编号 + 版本号
  页脚中: 页码 / 总页数 (如 1/10)
  页脚右: 生效日期

经典格式:
  顶部: [公司名] AC 电源线规格书 | 文件编号: ENG-SPEC-001 | V3.0
  底部: 第 1 页 / 共 10 页 | 生效日期: 2024-06-01 | 受控副本
```

### 4. 修订模式（审核文件时用）
```
操作路径: 审阅 → 修订
开启后: 所有修改会被标记（删除线 + 下划线 + 修订人 + 时间）
关闭后: 文档恢复正常

用途: 内审 / 客户审阅文件时保留修改痕迹
```

### 5. 文档保护
```
操作路径: 审阅 → 限制编辑
选项:
  - 仅允许在文档中进行此类编辑: "填写窗体"
  - 启动强制保护 → 设置密码
用途: 文件发放后防止误改
```

### 6. 模板（**最高效**）
```
操作路径: 文件 → 另存为 → 选择"Word 模板 (*.dotx)"
模板内容建议:
  - 公司 Logo
  - 统一的页眉页脚
  - 标准的标题样式
  - 文件编号位置
  - 受控章位置

好处: 所有新文件都用这个模板, 格式统一
```

## 三、Excel 必学 8 个技能

### 1. 数据透视表（**最强大工具**）

**用途**：从大量明细数据快速分类汇总。

**操作步骤**：
```
1. 选中数据区
2. 插入 → 数据透视表
3. 选择字段:
   - 行: 部门 / 产品 / 月份
   - 值: 数量 / 不良数 (求和 / 计数)
   - 列: 月份
4. 自动生成汇总表

例子: 统计各部门每月文件发放数量
```

**文控应用**：
- 文件发放数量按部门/月份汇总
- 不合格品数量按类型/月份汇总
- 内审不符合项按部门/条款汇总

### 2. VLOOKUP（**跨表查数据**）

**语法**：`=VLOOKUP(查找值, 表格区域, 列号, 0)`

**例子**：
```
=VLOOKUP(A2, 文件清单!A:F, 4, 0)
  ↑↑↑↑    ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑  ↑  ↑
  文件编号 文件总清单A到F列    返回第4列 精确匹配
           (编号、名称、版本、生效日期、状态、备注)
```

**文控应用**：
- 在借阅登记表里自动显示文件名称
- 在不合格报告里自动显示客户名称

### 3. 条件格式

**操作步骤**：
```
开始 → 条件格式 → 新建规则 → 使用公式

例子 1: 高亮过期日期
  公式: =TODAY()>B2
  格式: 红色填充

例子 2: 高亮"未关闭"CAPA
  公式: =$D2="未关闭"
  格式: 黄色填充
```

### 4. 冻结窗格

**操作步骤**：`视图 → 冻结窗格 → 冻结首行`（或冻结首列）

**文控应用**：文件总清单滚动时表头不动。

### 5. 数据验证（下拉）

**操作步骤**：`数据 → 数据验证 → 序列`

**例子**：状态列只允许"受控 / 非受控 / 作废 / 保密"

### 6. COUNTIF / SUMIF

**语法**：
```
COUNTIF(范围, 条件) - 计数
SUMIF(范围, 条件, 求和范围) - 求和

例子:
=COUNTIF(D:D, "受控")  ← 统计受控文件数量
=SUMIF(A:A, "AC电源线", B:B) ← 统计 AC 电源线数量
```

### 7. 文本函数

```
LEFT(A1, 3)   ← 取左 3 位 (部门代码)
RIGHT(A1, 6)  ← 取右 6 位 (版本号)
LEN(A1)       ← 字符串长度
CONCATENATE(A1, "-", B1) ← 拼接 (如 "QA-SOP-005")
```

### 8. 柏拉图制作

```
步骤:
1. 数据按"不良数"降序排列
2. 添加"累计%"列 (累加/总和)
3. 插入 → 图表 → 柱形图 (主轴: 不良数; 次轴: 累计%)
4. 把累计%改成折线图

公式: 累计% = SUM($B$2:B2) / SUM($B$2:$B$10)
```
